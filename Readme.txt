Forward Osmosis Performance Model

This model (developed using LabVIEW) predicts the performance of forward osmosis membrane modules based on the input salinities and flows. The model divides the membrane module in small sections, and it creates the performance profile across the module.

System Requirements:

Windows 7 SP1 or later - Mac OS 10.14 or later
1 GB RAM
1 GB Disk space

Installation - Windows:
Unzip file FO_Model_WIndows_Installer.zip
Run file: FO_model_install.exe and follow on-screen instructions

Installation - Mac OS:
Unzip file FO_Model_Mac.zip
Install LabVIEW Runtime: LabVIEW_Runtime_2020.dmg
Open folder FO_Model and Run app: FO_Model
